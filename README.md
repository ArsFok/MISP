# MISP

Есть 2 вариант списать:
Первый:
Через команду "cat название файла", у вас в терминале появится текст и вы просто перепечатываете то что там в новый файл.
Второй: 
Скинуть мне фото вашего билета в тг и подождать минут 5-7. После в папке ~/MISP$ вбить команду git pull и в папке MISP появится папка с вашим ФИО.
